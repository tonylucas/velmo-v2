# Délais de livraison

Le délai de livraison, une fois la commande préparée et expédiée, est de :

- France : J+2 (environ 2 jours ouvrés)
- Union européenne : J+5
- Reste du monde : J+10

Chaque maillot collector étant une pièce unique, la préparation soignée peut
ajouter 24 à 48 heures avant l'expédition.
