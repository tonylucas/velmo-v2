# Sécurité des paiements

Tous les paiements passent par un prestataire certifié et le protocole 3D Secure.
Velmo ne stocke jamais votre numéro de carte bancaire.

Si vous recevez un message vous demandant vos identifiants ou votre code de
carte, ne répondez pas : il s'agit d'une tentative de fraude.
